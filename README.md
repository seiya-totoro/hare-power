# 晴れパワー送受信機

2人だけで遊べる、スマホ向けのミニWebアプリです。

相手に専用URLを送ると、お互いに「億晴れパワー」とメッセージを送り合えます。LINEやInstagram DMで共有しやすい、青空・雲・太陽のやわらかいカード風デザインです。

## 概要

- 2人用の秘密URLで遊ぶ晴れパワー送受信アプリ
- どちらの人も送信と受信ができる
- 送る晴れパワーは「億晴れパワー」単位
- メッセージ付きで送信できる
- 相手から届いた晴れパワーを大きく表示
- やりとり履歴を表示
- 履歴は手動で削除できる
- 新しいURLを作れば、別の部屋として最初から遊べる

## URLの仕組み

このアプリは、1つの `room` に2人が入る形です。

```text
あなた用URL: index.html?side=a&room=秘密ID
相手用URL:   index.html?side=b&room=秘密ID
```

`room` が同じなら同じ部屋につながります。`side=a` と `side=b` は、どちら側の人かを分けるための印です。

URLを知っている人だけが開ける簡易的な秘密部屋なので、URLはSNSに公開せず、LINEやInstagram DMで直接送る想定です。

## 使い方

1. アプリのURLを開く
2. 自動であなた用URLが作られる
3. **相手用URLをコピー** を押す
4. コピーしたURLを相手に送る
5. お互いに名前、晴れパワー、メッセージを入力して送る
6. 相手から届いた晴れパワーは画面上のカードに表示される

## 主なボタン

- **晴れパワーを送る☀️**  
  入力した億晴れパワーとメッセージを相手に送ります。

- **相手用URLをコピー**  
  もう1人に送るためのURLをコピーします。

- **新しいURLを作る**  
  新しい部屋を作ります。前のテスト履歴とは別で遊び直せます。

- **履歴を消す**  
  今の部屋のやりとり履歴を削除します。同じ部屋を開いている相手側からも消えます。

## ファイル構成

```text
index.html   画面のHTML
style.css    デザインとアニメーション
script.js    Firebase連携とアプリの動き
README.md    この説明
```

## メモ

データの送受信には Firebase Realtime Database を使っています。GitHub Pagesのような静的ホスティングでも、2人の画面を同期して遊べます。
